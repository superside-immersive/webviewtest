# Assets para Página 1: AR SLAM

Colocá aquí los siguientes archivos:

## Modelo 3D
- `character.vrm` — Modelo VRM del personaje (ej: finalsnoo.vrm del repo slam)

## Animaciones (formato FBX, rig Mixamo)
- `idle.fbx` — Animación de reposo/idle en loop
- `jump.fbx` — Animación de salto (one-shot)

## Sonidos (formato MP3 o WAV)
- `sound1.mp3` — Sonido para botón 1
- `sound2.mp3` — Sonido para botón 2
- `sound3.mp3` — Sonido para botón 3

## Motor 8th Wall
Además, necesitás copiar la carpeta `xr-standalone/` del repo `superside-immersive/slam` 
a `page1-ar-slam/xr-standalone/`

## Preview local
Si no copiás `xr-standalone/`, la página ahora cae a un preview 3D local sin SLAM.
Para AR real en móvil seguís necesitando `xr-standalone/` y abrir por `https://`.
